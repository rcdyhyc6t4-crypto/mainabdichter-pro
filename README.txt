MAINABDICHTER PRO V1

Enthalten:
- GPS-Standortübernahme
- Kunden- und Objekterfassung
- Horizontalsperre / Flächensperre
- automatische Material- und Arbeitszeitberechnung
- Baustelleneinrichtung inkl. An-/Abfahrt 269 € netto
- Mindest-, Angebots-, Optimal- und Verhandlungspreis
- lokale Speicherung
- Drucken / PDF über Safari
- Offline-Grundfunktion nach Installation

Zum Testen am Rechner:
1. Ordner entpacken.
2. In diesem Ordner einen kleinen Webserver starten, z. B.:
   python3 -m http.server 8000
3. Browser öffnen: http://localhost:8000

Für das iPhone:
Die Dateien müssen einmal auf einem HTTPS-Webspace veröffentlicht werden.
Danach in Safari öffnen und über 'Teilen' > 'Zum Home-Bildschirm' installieren.
GPS funktioniert nur über HTTPS oder localhost.
